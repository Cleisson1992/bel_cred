# Brainstorm de Design - BelCred Landing Page

## Contexto
Landing page para captura de leads de empréstimo consignado. Objetivo: converter visitantes em leads qualificados. Público: profissionais CLT, pensionistas, concursados e aposentados.

---

## Abordagem 1: Modernismo Corporativo com Ênfase em Confiança
**Probabilidade: 0.08**

### Design Movement
Modernismo corporativo com influências de design financeiro contemporâneo (inspirado em fintechs bem-sucedidas).

### Core Principles
1. **Confiança através de Clareza**: Hierarquia visual muito clara, sem ambiguidades
2. **Minimalismo Funcional**: Apenas elementos que servem a um propósito específico
3. **Profissionalismo Elegante**: Linhas limpas, espaçamento generoso, tipografia sofisticada
4. **Segurança Visual**: Uso de cores sólidas, sem gradientes excessivos

### Color Philosophy
- **Primária**: Azul profundo (#1e40af) - transmite confiança, segurança, profissionalismo
- **Secundária**: Amarelo ouro (#f59e0b) - destaque, energia, otimismo (do logotipo)
- **Neutros**: Cinzas sofisticados (fundo branco, textos em cinza-escuro)
- **Acentos**: Verde suave (#10b981) para CTAs positivas (aprovação, sucesso)

### Layout Paradigm
- **Hero Section**: Assimétrico com imagem à direita, conteúdo à esquerda
- **Seções**: Alternância entre fundo branco e cinza muito claro (quase branco)
- **Grid**: Estrutura de 12 colunas, mas com blocos assimétricos
- **Modalidades**: Cards em grid 2x3 com ícones minimalistas

### Signature Elements
1. **Linhas Divisórias Horizontais**: Separadores sutis em amarelo ouro
2. **Ícones Geométricos**: Ícones de linha (stroke) em azul, sem preenchimento
3. **Badges de Confiança**: Pequenos selos com checkmarks em verde

### Interaction Philosophy
- Transições suaves (200-300ms) em hover
- Botões com efeito de elevação (box-shadow) ao passar o mouse
- Formulário com validação em tempo real, feedback visual claro

### Animation
- Fade-in suave ao scroll (0.6s de duração)
- Ícones que ganham cor ao hover (azul → amarelo)
- Números de estatísticas que contam (contador animado)

### Typography System
- **Display**: "Playfair Display" (serif elegante) para títulos principais
- **Heading**: "Inter" 700 para subtítulos
- **Body**: "Inter" 400/500 para corpo de texto
- **Hierarquia**: 48px → 32px → 24px → 16px → 14px

## Abordagem 2: Dinâmico e Acessível (Design Inclusivo)
**Probabilidade: 0.07**

### Design Movement
Design inclusivo com influências de movimento de acessibilidade digital e design humanista.

### Core Principles
1. **Acessibilidade em Primeiro Lugar**: Contraste WCAG AA+, tipografia legível, navegação clara
2. **Inclusão Visual**: Representação diversa, ícones que falam para todos
3. **Dinamismo Controlado**: Movimento que agrega valor, não distrai
4. **Transparência**: Informações claras sobre processo, taxas, requisitos

### Color Philosophy
- **Primária**: Azul vibrante (#0066cc) - energia, confiança
- **Secundária**: Amarelo quente (#ffc107) - atenção, otimismo
- **Terciária**: Laranja suave (#ff9800) - ação, movimento
- **Neutros**: Preto real (#000) para texto, branco (#fff) para fundo
- **Acessibilidade**: Alto contraste em todos os textos

### Layout Paradigm
- **Hero**: Full-width com imagem de fundo (com overlay para legibilidade)
- **Seções**: Blocos de conteúdo com bordas arredondadas (16px)
- **Modalidades**: Cards com ícones grandes, descrição clara
- **Depoimentos**: Carrossel com navegação clara (setas grandes)

### Signature Elements
1. **Ícones Grandes e Coloridos**: Ícones de 64px+ com cores variadas
2. **Botões com Ícones**: CTAs com ícones à direita (seta, play, etc.)
3. **Badges Coloridas**: Diferentes cores para diferentes modalidades

### Interaction Philosophy
- Feedback imediato em todas as interações
- Animações que guiam a atenção (pulse, bounce suave)
- Formulário com dicas contextuais (tooltips acessíveis)

### Animation
- Ícones que "pulam" ao scroll (bounce suave)
- Transição de cores em hover (0.3s)
- Cards que deslizam de baixo para cima ao aparecer

### Typography System
- **Display**: "Poppins" 700 (sans-serif moderno) para títulos
- **Heading**: "Poppins" 600 para subtítulos
- **Body**: "Poppins" 400/500 para corpo
- **Hierarquia**: 56px → 40px → 28px → 18px → 16px

## Abordagem 3: Premium e Sofisticado (Luxury Fintech)
**Probabilidade: 0.09**

### Design Movement
Design de luxo contemporâneo com influências de premium fintech (Nubank, Revolut).

### Core Principles
1. **Sofisticação Subtil**: Detalhes refinados, não óbvios
2. **Espaço Negativo Generoso**: Muito branco, pouco conteúdo por seção
3. **Tipografia Elegante**: Fontes serif para títulos, sans-serif para corpo
4. **Gradientes Sutis**: Apenas em áreas estratégicas, nunca excessivos

### Color Philosophy
- **Primária**: Azul escuro (#0f172a) - sofisticação, profundidade
- **Secundária**: Amarelo ouro (#d97706) - luxo, destaque (do logotipo)
- **Terciária**: Branco com toque de cinza (#f8fafc) - pureza
- **Acentos**: Gradiente sutil azul → roxo para CTAs especiais

### Layout Paradigm
- **Hero**: Assimétrico com muito espaço negativo, imagem pequena e refinada
- **Seções**: Blocos grandes com muito padding (80px+)
- **Modalidades**: Cards minimalistas, apenas ícone + título + descrição breve
- **Depoimentos**: Layout em coluna única, citações grandes e elegantes

### Signature Elements
1. **Linhas Finas**: Divisores em ouro muito suave
2. **Ícones Minimalistas**: Ícones de stroke fino, elegantes
3. **Tipografia Display**: Títulos em serif grande e espaçado

### Interaction Philosophy
- Transições muito suaves (400-600ms)
- Hover effects sutis (mudança de cor suave, não drástica)
- Formulário com estilo minimalista (sem bordas visíveis, apenas underline)

### Animation
- Fade-in muito lento ao scroll (1s+)
- Ícones que ganham brilho suave (glow effect)
- Números que contam com transição suave

### Typography System
- **Display**: "Cormorant Garamond" (serif elegante) para títulos principais
- **Heading**: "Cormorant Garamond" 600 para subtítulos
- **Body**: "Lato" 300/400 para corpo (sans-serif leve)
- **Hierarquia**: 64px → 48px → 32px → 18px → 14px

---

## Decisão Final
**Escolhida: Abordagem 1 - Modernismo Corporativo com Ênfase em Confiança**

### Justificativa
Para uma landing page de empréstimo consignado, a **confiança é o fator crítico**. O Modernismo Corporativo oferece:
- Clareza visual que reduz fricção no formulário de captura
- Profissionalismo que aumenta a credibilidade
- Hierarquia clara que guia o usuário para o CTA
- Paleta de cores que combina com o logotipo existente
- Acessibilidade natural através de minimalismo

Esta abordagem é ideal para **tráfego pago** porque:
1. Reduz bounce rate (design limpo não intimida)
2. Aumenta conversão (CTAs claros em verde)
3. Transmite segurança (cores sólidas, sem excessos)
4. Funciona bem em mobile (hierarquia responsiva)
